# 

## Встраиваемый безкорпусный USB хаб

Компактный 4-портовый USB-hub для постоянного подключения USB-устройств робота к встроенному зарядному устройству. Порт верхнего уровня имеет разъем USB-C. Выходные порты можно запаять как вам удобно: либо на индивидуальные разъемы с шагом 2.54, либо в соответсвии со стандартами ПК один разъем на два порта ([10 pin usb connector](https://www.google.ru/search?q=10+pin+usb+motherboard+connector&newwindow=1&sca_esv=ad6c775310e3a212&udm=2&biw=1632&bih=968&aic=0&sxsrf=ANbL-n6g4Ry-VCOGpTZLcD4XbuAmTV0oSw%3A1770105939725&ei=U6yBaeD5K9SmwPAP3vyF6AQ&ved=0ahUKEwig74G-7rySAxVUExAIHV5-AU0Q4dUDCBI&uact=5&oq=10+pin+usb+motherboard+connector&gs_lp=Egtnd3Mtd2l6LWltZyIgMTAgcGluIHVzYiBtb3RoZXJib2FyZCBjb25uZWN0b3IyBRAAGIAESJRPUN4BWIxNcAd4AJABAJgBLKAB9gOqAQIxMrgBA8gBAPgBAZgCE6ACqASoAgrCAgQQABgewgIGEAAYCBgewgIKECMYJxjJAhjqAsICBhAAGAcYHsICCBAAGAcYCBgemAMEiAYBkgcCMTmgB4MNsgcCMTK4B4sEwgcGMTAuNC41yAcjgAgA&sclient=gws-wiz-img)).

### Технические характеристики

* **Микрочип** FE1.1s
* **Интерфейсы:**
  * Вход: USB 2.0, USB-C, TVS защита
  * Выход: 4 USB 2.0 
* Светодиоды для индикации активности
* Совместимость со стандартным 10-pin разъемом или индивидуальныvb XH2.54 
* Размеры:
  * Плата: 41x31мм
  * монтажные отверстия: D2.5 35x25мм
