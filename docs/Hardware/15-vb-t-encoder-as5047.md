#

## Т-энкодер. Датчик угла для мотор-колес на основе AS4047p.

Эта компактная отладочная плата построена на базе AS5047P — высокопроизводительного 14-битного магнитного поворотного энкодера. Она предоставляет данные об абсолютном угловом положении с исключительным разрешением (16 384 позиции на оборот) через выход SPI.

Плата T-Encoder, специально разработана и представляет собой идеальное решение для простого преобразования мотор-колёс от гироскутеров в двигатели с управлением моментом для робототехники, предлагая простое и надёжное решение обратной связи для FOC-приложений.

Датчик состоит из двух частей и поставляется в разобранном виде. Нижнюю ножку необходимо припаять самостоятельно (при необходимости).

### Технические характеристики
* Датчик: AS5047P ([Документация](/files/pdf/AS5047_datasheet.pdf))
* Интерфейс: SPI
* Размеры
  * Плата с датчиком: D12мм, высота с датчиком 2.6мм
  * Ножка: 8x5мм
