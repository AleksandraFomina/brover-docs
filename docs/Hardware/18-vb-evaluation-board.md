# 
## Отладочная плата для модуля VBCores32G4 


Эта плата предназначена для оценки возможностей модуля [VBCore VB32G4](vbcore-vb32g4.md). Она идеально подходит для образовательных целей и отладки, поскольку имеет специальную контактную площадку, которая исключает необходимость пайки модуля. Так же на ней можно разместить бредборды не требующие пайки.

Используется в нашем обучающем **[курсе по электронике](https://edu.vbcores.ru/1-programming-mс/intro)** для начинающих.

### Характеристики

- Беспаечная платформа для модуля VBCore
- Порт программирования
- Порт CAN
- Разъём внешнего питания DC 2.1
- I2C EEPROM объёмом 256 Кбит
- Управление напряжением
- Макетная печатная плата
- Совместимость с распиновкой Arduino
- Совместимость с распиновкой Nucleo64
- Размеры:
  - Плата: 100x130x15 mm
  - Крепежнык отверстия: D3, 90x120мм
