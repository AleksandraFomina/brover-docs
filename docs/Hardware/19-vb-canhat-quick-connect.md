# 
## Комутационная плата для питания


Эта плата предназначена для удобного подключения внешних устройств к сборке микрокомпьютера Raspberry и CAN-Адаптера.  Позволяет независимо комутировать питание внешнего устройства, защищает линию питания предохранителем и позволяет  подключать CAN шину стандартными для всей системы VBCores разъемами XH2.54

Используется для соревнований и в нашем обучающем  **[курсе по работе с коммуникациями по CAN](https://edu.vbcores.ru/2-communication/intro/)** для начинающих.

### Характеристики
- Быстросоединяемые разъемы:
  - XH2.54 для CAN линий
  - XT30 для линий питания
  - Предохранитель линии питания (по умолчанию 1A, MAX 10A)
  - Выключатель линии питания 
  - Индикация питания на входе
  - Индикация питания на выходе
- Размеры:
  - Плата: 65x56x15 mm
  - Крепежнык отверстия: D2.5, 58x49мм
