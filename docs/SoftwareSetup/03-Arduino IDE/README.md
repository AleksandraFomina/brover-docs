# Arduino IDE

При разработке модуля **VBCore** мы постарались сделать так, чтобы даже не самые опытные разработчики, владеющие навыками работы с **Arduino**, но никогда на занимавшиеся серьезной разработкой под микроконтроллеры **STM32**, смогли достигнуть весьма приемлемого результата в своих робототехнических проектах.

Поэтому  модули **VBCores** и большинство функциональных плат  позволяют работать с собой через **Arduino IDE**  с использованием библиотеки **STM32duino** ([https://github.com/stm32duino](https://github.com/stm32duino)).

При этом сохраняются все преимущества модулей, такие как встроенный **CAN FD**,  работа в широком диапазоне входных напряжений и быстродействие, и при этом вам становиятся доступна огромная база библиотек разработанных для **Arduino**.

Однако немного придется повозится с настройками. Если вас это заинтересовало, переходите к следующей странице.
